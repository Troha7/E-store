About project

This project is a fully Reactive application.

Project Stack:

  • Spring Boot
  • Spring Webflux
  • Spring Security
  • PostgreSQL
  • Spring Data R2DBC
  • Flyway
  • Thymeleaf (HTML, CSS)
  • Bootstrap
  • JavaScript (a little)

Testing:

This project tests API endpoints using a PostgreSQL test Docker container.

You can also use the Swagger-UI to test the Rest APIs.
